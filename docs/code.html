Kód felépítése:

Grafikus felület

A játékot python programozási nyelven írtuk, egy Tkinter nevű open source vizuális felület felhasználásával.

Csapat class

A játék legfőbb elemeit (a csapatokat) egy külön osztályba tároljuk el, ami tárolja az összes csapathoz tartozó információt és a csapat összes tagját.

Játékos class

A csapat tagjait külön típusként mentjük el, amiben eltároljuk a játékos nevét, statisztikáit, pozícióját és az általunk kiszámított összpontszámát.

Helyzet class

Az éles játék folyamán a helyzet (chance) osztály segítségével generálunk kommentárt az éppen folyó játékhoz. Ez a class menti el az éppen helyzetbe kerülő játékost és csapatát is. Ez a helyzet nem muszáj, hogy kapuralövés legyen. Lehet például sárgalap, vagy sérülés is. Hogyha valami változás történik valamelyik csapatban, akkor újrageneráljuk a helyzeteket, így lesz igazán változatos a mérkőzés. A helyzeteket a GUI classban generáljuk.

GUI class

A programkód legnagyobb része a GUI osztályban található, ahol meghatározzuk a játék összes elemét és azok helyét a képernyőn.
